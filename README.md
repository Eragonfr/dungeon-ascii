# Jeu

**THIS BRANCH IS A LINUX ONLY VERSION !**  
*Windows is supported through WSL*

Travail de classe.  
Objectif: Fabriquer un Dungeon Scroller.

## Build

Depend de ncurses.  

Utilisez Make, c'est bien make.  
Clonez le dépôt.  
```bash
git clone https://github.com/Eragonfr/dungeon-ascii.git
cd dungeon-ascii
make
```
